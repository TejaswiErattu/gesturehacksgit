# Muse Educator Pilot Landing Page

A compelling landing page for Muse's upcoming Educator Pilot Launch, designed to introduce the AI learning platform to educators and highlight how it supports meaningful learning through storytelling.

## Project Overview

This landing page transforms abstract concepts into emotional connections, helping educators understand how Muse can revolutionize their classrooms by moving beyond assignment completion to foster genuine curiosity and meaningful exploration.

## Key Features

- **Story-Driven Messaging**: Emphasizes Muse's core mission of learning through storytelling
- **Educator-Focused Content**: Specifically targets professors, instructors, and K-12 teachers
- **Authentic Testimonials**: Features realistic educator quotes showcasing real-world impact
- **Cohesive Brand Design**: Uses Muse's brand colors (purple, indigo, warm cream) and typography
- **Clear Call-to-Action**: Prominent sign-up for the educator pilot program
- **Responsive Design**: Optimized for all devices and screen sizes

## Design Elements

### Brand Colors
- Primary Purple: `#6B46C1`
- Secondary Indigo: `#4338CA`
- Warm Cream: `#FEF7E8`
- Accent Orange: `#EA580C`

### Typography
- Primary font: Georgia (serif) for warmth and readability
- Elegant, story-focused aesthetic that appeals to educators

### Content Sections
1. **Hero**: Powerful headline about transforming learning through story
2. **Value Proposition**: Clear explanation of why Muse matters for educators
3. **Features**: Three core benefits (Story-Driven Learning, Emotional Connection, Intentional Learning)
4. **Testimonials**: Two educator quotes demonstrating real impact
5. **Call-to-Action**: Prominent sign-up for early access

## How to View

### Option 1: Open directly
Double-click `index.html` to open it in your default browser.

### Option 2: Local server
Start a local server for development:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000 in your browser.
